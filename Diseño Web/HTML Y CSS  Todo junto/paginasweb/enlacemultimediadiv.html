Primeros Enlaces y como hacerlo

Para los enlaces se pone >a< luego en el atributo que es href luego se pone el enlace que puede ser interno como en una carpeta (primer enlace) y externo como Google (Segundo enlace).
Para determinar la carpeta se pone por ejemplo "prueba"(carpeta) luego la / y el archivo que termine en html y queres volver al archivo se pone ../ y el archivo y podes poner por ejemplo "../../../Index.html"
Luego para mandarte a otra pestaña al abrir el enlace se usa el atributo target="_BLANK"

Primeras Imagenes Y Videos

Para lograrlo se pone >img< y en el atributo src o source se pone el link de la imagen, para lograr el link de la imagen hay que buscar una apretar click derecho, luego en inspeccionar y copiamos el enlace que tiene la >img> y el source.
Otro atributo que se le puede poner a las imagenes (NO ES RECOMENDABLE POR QUE MODIFICA EL TAMAÑO DEL PNG Y ESO SE HACE CON CSS) son la de Width Y Height
Aca abajo te dejo una imagen (Externa) con Width 260px

Ahora paso a mostrar una imagen interna de la carpeta para poner la imagen descargada se pone >img< y en el atributo src va el link del archivo. Otro atributo que se le puede poner a las imagenes son la de Width Y Height
Aca abajo te dejo una imagen (Interna) o (descargada) con Height 300px

Otros atributos muy vitales a la hora de poner una imagen son los Atributos:
alt=""(es como un "título" cuando no esta la imagen)
title="" (Sirve para poner un título a la imagen, cuando el usuario pase el mouse por encima de la imagen)

Ahora pasamos a como poner videos en nuestra pagina web con >video src=""< y se cierra con >/video< luego ponemos el atributo controls para que se ejecute el video

Para los Audios y Videos se usa el mismo formato nomas cambia el >audio<
